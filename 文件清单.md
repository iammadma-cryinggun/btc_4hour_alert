## 🚀 快速部署

### 方式1: 使用一键部署脚本（推荐）
```bash
# 上传所有文件到服务器后
chmod +x deploy.sh
./deploy.sh
```

### 方式2: 手动部署
```bash
# 1. 安装依赖
pip3 install -r requirements_v411.txt

# 2. 运行程序
python3 BTC4小时策略预警v4.1.1_双信号版.py
```

### 方式3: 使用systemd（长期运行）
```bash
# 1. 使用deploy.sh创建服务
./deploy.sh

# 2. 启动服务
sudo systemctl start btc-physics-v411.service

# 3. 查看状态
sudo systemctl status btc-physics-v411.service
```
