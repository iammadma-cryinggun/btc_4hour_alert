## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

编辑 `.env` 文件：

```bash
# Telegram配置
TELEGRAM_TOKEN=你的Token
TELEGRAM_CHAT_ID=你的ChatID
TELEGRAM_ENABLED=True

# 代理配置（如果需要）
PROXY_ENABLED=True
PROXY_HOST=127.0.0.1
PROXY_PORT=15236
```

### 3. 运行系统

```bash
python BTC4小时策略预警v4.2_数学家完整版.py
```

## 🛠️ 故障排查

### 问题1: 无法接收Telegram通知

**检查**:
1. Token和Chat ID是否正确
2. 代理是否工作
3. 网络是否正常

### 问题2: DXY数据获取失败

**解决**: 系统会自动降级为无DXY模式，不影响核心功能

### 问题3: 系统启动后立即退出

**检查**:
1. 查看日志文件 `physics_alert_v4_2_math.log`
2. 确认所有依赖已安装
3. 检查Python版本
